## 2. What is JavaScript?

## 3. Name three problems Git & GitHub solve?

Working collaboratively, managing code versions, autmating software deployment.

## 4. What happens when you `fork` a repository?

You make your own copy.

## 5. What happens when you clone a repository?

It downloads a copy to your local machine, with a link to the remote repository.

## 6. What is a Pull Request?

When you request that some changes you made to a code base to be integrated into the the existing code.
